Verifying if a .txt AMARES Output File can be Processed

Contents

  • Output
  • Child Functions
  • Notes

Input

inp: is a .txt AMARES output file

Output

bool: is 1 if the file can be processed, or it is 0 if the file cannot be processed

Child Functions

isdatfile: checks if an input is a readible file

loadJmruiAmares: parses the .txt output from AMARES in JMRUI into a structure array

function bool = validateAMARES(inp)

    bool = 1;

    if nargin == 0
        bool = 0;
        disp('Error: No input entered')
    end

    boolf = isdatfile(inp);

    if boolf == 0
        bool = 0;
    end

    if boolf == 1
        try
            loadJmruiAmares(inp);
        catch
            disp('Error: Invalid file entered');
            bool = 0;
        end
    end

end
