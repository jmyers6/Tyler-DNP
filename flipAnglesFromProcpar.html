Input

proc: is a procpar file from an NMR experiment

Output

flip1: is an array that contains the contents of the flip1 parameter in the procpar file

Child Function

readprocpar: parses the contents of a procpar file into a structure array

function flip1 = flipAnglesFromProcpar(proc)

Checking the Number of Inputs

    procbool = 1;

    if nargin < 1
        procbool = 0;
    end

Parsing the Procpar File

    if procbool == 0
        procpar = readprocpar;
    else
        procpar = readprocpar(proc);
    end

Extracting the flip1 Parameter

    flip1a = procpar.flip1;
    flip1 = transpose(flip1a);

end
